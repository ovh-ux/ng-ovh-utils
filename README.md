## Installation

### NPM

```sh
$ npm install @ovh-ux/ovh-utils-angular --save # or yarn add @ovh-ux/ovh-utils-angular
```

## Get the sources

```sh
$ git clone https://github.com/ovh-ux/ovh-utils-angular.git
$ cd ovh-utils-angular
$ npm install # or yarn
```
